## ğŸ“¦ How to Use

1.  Download or copy the code into an `index.html` file.
2.  Open it in any modern web browser (Chrome, Safari, Edge, Firefox).
3.  **Configure your workout:**
    * **Work/Rest:** Adjust seconds using the +/- buttons.

4.  Press **START** and get moving!

> **Note:** The "Wake Lock" feature (keeping screen on) requires the page to be visible and active. It works best on Chrome, Edge, and Safari on iOS/Android.
